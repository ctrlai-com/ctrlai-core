
# Ctrl AI: Custom Vocabulary

**Version:** 1.0
**Date:** 2025-02-17

## 1. Introduction

This document defines the custom vocabulary for the Ctrl AI protocol. This vocabulary extends [Schema.org](https://schema.org/) with terms specific to Ctrl AI, ensuring interoperability and a shared understanding of the data.

**Base URI:** `https://ctrlai.com/schema/` (This is a placeholder; you'll need to host this vocabulary at a publicly accessible URL).
